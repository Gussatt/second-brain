# Agent Instructions

This vault is organized into specific directories. When working within this vault, refer to the following structure:

- **`00 Inbox/`**: Raw inputs and immutable source documents.
- **`01 Updates/`**: Daily research updates by topic.
- **`02 Daily/YYYY/MM/`**: Daily notes logging session activity.
- **`03 Meetings/`**: Call/meeting notes and transcripts.
- **`04 People/`**: Detailed snapshots of relationships, family, and history.
- **`05 Projects/`**: Active project folders.
- **`06 Wiki/`**: The knowledge powerhouse with slug-named pages (`pages/`), `index.md`, `log.md`, and `overview.md`.
- **`07 Summaries/`**: AI-generated summaries of external content.

**Key Rules:**
- Do not add `# Title` headings to notes.
- Use `unread: true` in the frontmatter of any new or modified note.
- In `06 Wiki/`, all page filenames must use lowercase slugs with hyphens (e.g., `machine-learning.md`).
- Always cite factual claims using footnotes and `[[source-slug]]`.
- For extensive guidelines, refer to `GEMINI.md`.
